# Инструкция по открытию devcontainer
1. обязательно наличие Docker на компьютере, его можно установить с официального сайта 
2. далее необходимо скачать расширение "Dev Containers":  открыть VS Code, перейти в раздел «Расширения» (Ctrl+Shift+X или Cmd+Shift+X), найти «Dev Containers» и установить расширение с ID: ms-vscode-remote.remote-containers
3. я открывала hydrolib-soft с этим расширением с помощью поисковой строки командой ">DevContainers: Open Folder in Container..." -> выбрала папку HydroLib-soft 
4. подождать пока папка откроется в контейнере и готово
* по окончании проверить, чтобы в левом нижнем углу была плашка "Dev Container: Hydrolib @ desktop-linux"
